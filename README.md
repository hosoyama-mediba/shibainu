# shibainu

## Usage

### Installation

```sh
git clone https://github.com/hosoyama-mediba/shibainu.git
cd shibainu
npm install
```

### Execution

```sh
node index.js -u http://m.yahoo.co.jp/ -o ss/ -f `date "+%Y%m%d%H%M"`.png
```

### Options

```
  -u url
  -o output directory
  -f file name
  -a user agentt
```
